# 🎤 Голосовые команды - Краткая справка

## 🟢 Для ВСЕХ пользователей (Employee, Supervisor, Admin, Superadmin)

### Навигация
```
🎤 "Покажи календарь"          → Открывает календарь
🎤 "Show calendar"

🎤 "Мои отпуска"                → Список отпусков
🎤 "My leaves"

🎤 "Мои задачи"                 → Список задач
🎤 "My tasks"

🎤 "Мой профиль"                → Настройки профиля
🎤 "My profile"

🎤 "Посещаемость"               → Чек-in/чек-out
🎤 "Attendance"
```

### Информационные запросы
```
🎤 "Сколько у меня отпуска?"    → Показывает баланс
🎤 "Кто сегодня на работе?"     → Список присутствующих
🎤 "Кто в отпуске?"             → Список людей в отпуске
🎤 "Когда [Имя] в отпуске?"     → Отпуск конкретного человека
```

### Бронирование отпусков
```
🎤 "Хочу взять отпуск с 10 по 15 марта"
🎤 "Забронировать больничный на завтра"
🎤 "I want vacation from March 10 to 15"
```

## 💡 Советы по использованию

### ✅ Работает хорошо:
- Четкое произношение
- Короткие команды
- Ключевые слова (календарь, задачи, профиль)

### ❌ Может не сработать:
- Слишком длинные фразы
- Фоновый шум
- Нечеткое произношение

### 🎯 Лучшие практики:
1. Говорите четко и спокойно
2. Используйте ключевые слова из списка
3. Дождитесь индикации прослушивания (синяя рамка)
4. Пауза 1 секунда = автоотправка
