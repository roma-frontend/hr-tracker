# 🤖 AI Assistant - Полное руководство

## 📋 Обзор

AI ассистент теперь полностью интегрирован с системой ролей и предоставляет контекстно-зависимые возможности для каждого типа пользователя.

## 🎯 Ключевые возможности

### 1. **Роле-зависимые разрешения**
Ассистент автоматически определяет роль пользователя и предоставляет только доступные функции:

- **👤 Employee (Сотрудник)**: Личные данные, отпуска, задачи
- **👔 Supervisor (Супервайзер)**: Управление командой, одобрение отпусков
- **👨‍💼 Admin (Администратор)**: Управление организацией, аналитика, отчеты
- **🔴 Superadmin (Суперадмин)**: Полный доступ ко всем организациям и системе

### 2. **Голосовое управление**
Используйте кнопку микрофона 🎤 в чате для голосовых команд:

#### Примеры команд на русском:
```
"Покажи календарь"
"Открой мои задачи"
"Мой профиль"
"Список сотрудников"
"Аналитика"
"Покажи отчеты"
```

#### Examples in English:
```
"Show calendar"
"Open my tasks"
"My profile"
"Employees list"
"Analytics"
"Show reports"
```

### 3. **Умная навигация**
AI автоматически определяет намерение и перенаправляет на нужную страницу:

- ✅ Распознает команды на русском и английском
- ✅ Работает через голос и текст
- ✅ Проверяет разрешения перед навигацией
- ✅ Показывает только доступные функции

## 🗂️ Доступные функции по ролям

### 🟢 Employee (Сотрудник)
```
✅ Просмотр календаря
✅ Мои отпуска и баланс
✅ Бронирование отпусков
✅ Мои задачи
✅ Профиль и настройки
✅ Посещаемость (чек-in/чек-out)
✅ Просмотр команды (кто доступен)
```

**Начальные подсказки:**
- "Покажи мой календарь"
- "Сколько у меня осталось отпуска?"
- "Кто сегодня на работе?"
- "Мои задачи"

### 🟡 Supervisor (Супервайзер)
Все возможности Employee +
```
✅ Одобрение отпусков команды
✅ Просмотр посещаемости команды
✅ Оценка сотрудников
✅ Назначение задач
```

**Начальные подсказки:**
- "Показать заявки на отпуск"
- "Кто из команды опоздал сегодня?"
- "Посещаемость команды"
- "Создать задачу"

### 🟠 Admin (Администратор)
Все возможности Supervisor + Employee +
```
✅ Управление сотрудниками
✅ Аналитика и отчеты
✅ Одобрение заявок на присоединение
✅ Настройки организации
✅ Экспорт данных
```

**Начальные подсказки:**
- "Показать аналитику"
- "Список всех сотрудников"
- "Заявки на присоединение"
- "Отчет по отпускам"

### 🔴 Superadmin (Суперадмин)
Все возможности Admin + Supervisor + Employee +
```
✅ Управление всеми организациями
✅ Одобрение создания организаций
✅ Мониторинг безопасности
✅ Stripe подписки и платежи
✅ Системные настройки
```

**Начальные подсказки:**
- "Все организации"
- "Безопасность"
- "Stripe подписки"
- "Заявки на создание организаций"

## 📱 Интеграция с существующими функциями

### Бронирование отпусков

**🆕 НОВАЯ ФУНКЦИЯ: Интеллектуальная проверка конфликтов!**

AI ассистент теперь автоматически:
- ✅ Проверяет конфликты с отпусками коллег из вашего отдела
- ✅ Предлагает альтернативные даты при обнаружении конфликтов
- ✅ Запрашивает подтверждение перед отправкой заявки
- ✅ Анализирует загрузку отдела (если >30% в отпуске - рекомендует другие даты)

**Пример работы с конфликтом:**
```
👤 "Хочу отпуск с 10 по 15 марта"

🤖 "Проверяю доступность... ⚠️ Иван из вашего отдела уже в отпуске 12-14 марта.
    
    📅 Рекомендую альтернативные даты:
    • 17-22 марта (без конфликтов)
    • 24-29 марта (без конфликтов)
    
    Какие даты предпочитаете?"

👤 "17-22 подходит"

🤖 "Отлично! Отправляю запрос... ✅"
```

### Бронирование отпусков (базовые команды)
AI может напрямую бронировать отпуска:
```
Пользователь: "Хочу взять отпуск с 10 по 20 марта"
AI: Создает карточку с деталями отпуска
Пользователь: Подтверждает нажатием кнопки
→ Отпуск отправляется на одобрение
```

### Информация о команде
```
Пользователь: "Кто сегодня на работе?"
AI: Показывает список сотрудников с их статусами:
- 🟢 На работе
- 🏖️ В отпуске
- 🤒 На больничном
- 📅 На встрече
```

### Навигация
```
Любой запрос типа:
"Покажи X" / "Открой Y" / "Перейди к Z"
→ AI автоматически определяет страницу и перенаправляет
```

### Добавление новых возможностей

Чтобы добавить новую функцию в `src/lib/aiAssistant.ts`:

```typescript
{
  id: 'my_feature',
  name: 'My Feature',
  description: 'Description of what it does',
  requiredRole: ['admin', 'superadmin'], // Кто имеет доступ
  keywords: ['feature', 'функция', 'my feature', 'моя функция'],
  action: '/my-feature' // URL для навигации
}
```

## 🎨 Кастомизация подсказок

Подсказки автоматически адаптируются под роль пользователя.

Редактируйте в `src/lib/aiAssistant.ts`:
```typescript
export function getRoleSuggestions(role: UserRole): string[] {
  const allSuggestions: Record<UserRole, string[]> = {
    employee: [
      "Покажи мой календарь",
      // ... добавьте свои
    ],
    // ...
  };
  return allSuggestions[role];
}
```
