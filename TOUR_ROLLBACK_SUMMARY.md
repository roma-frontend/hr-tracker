### 4. **Упрощена функция updateTargetPosition**
```typescript

// СТАЛО: ~10 строк - простая логика

const updateTargetPosition = useCallback(() => {
  const step = steps[currentStep];
  if (!step?.target) return;

  const element = document.querySelector(step.target);
  if (element) {
    const rect = element.getBoundingClientRect();
    setTargetRect(rect);
    positionTooltip(rect, step.placement || "bottom");
  }
}, [currentStep, steps]);
```

## 🎯 Почему откат был необходим

### Проблемы добавленного функционала:
1. **Постоянное мигание** - из-за флагов isPositioned/isScrolling
2. **Рекурсивные обновления** - скролл вызывал скролл
3. **Задержки** - debounce создавал лаги
4. **Сложность** - слишком много логики
5. **Не работало на больших экранах** - баги с позиционированием

### Решение:
- Вернуться к простой рабочей версии
- При необходимости добавить минимальный скролл позже
- Не усложнять то, что работает
