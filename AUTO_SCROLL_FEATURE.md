# 📜 Автоматический скролл в Tour Guide - Готово!

## ✅ Что было добавлено

Теперь Tour Guide автоматически плавно прокручивает страницу к элементам, которые показывает, чтобы они всегда были в видимой области экрана.

---

## 🎯 Как это работает

### 1. **Проверка видимости элемента**
```typescript
// Проверяет, виден ли элемент полностью на экране
const isElementVisible = (
  rect.top >= 0 &&
  rect.left >= 0 &&
  rect.bottom <= window.innerHeight &&
  rect.right <= window.innerWidth
);
```

### 2. **Автоматический скролл**
Если элемент не виден или частично скрыт:
```typescript
// Вычисляет позицию для центрирования элемента
const scrollToY = absoluteElementTop - (window.innerHeight / 2) + (rect.height / 2);

// Плавно прокручивает к элементу
window.scrollTo({
  top: Math.max(0, scrollToY),
  behavior: 'smooth' // Плавная анимация
});
```

### 3. **Таймаут для анимации**
```typescript
// Ждет завершения анимации скролла (600ms)
setTimeout(() => {
  // Обновляет позицию tooltip после скролла
  const updatedRect = element.getBoundingClientRect();
  positionTooltip(updatedRect, placement);
}, 600);
```

## 🎨 Особенности

### ✅ Плавный скролл:
- Анимация `behavior: 'smooth'`
- Длительность: ~600ms
- Автоматическое центрирование элемента

### ✅ Умное поведение:
- Скролл только если элемент не виден
- Центрирование по вертикали и горизонтали
- Не прокручивает, если элемент уже на экране

### ✅ Адаптивность:
- Работает на маленьких экранах
- Работает на мобильных устройствах
- Работает при любой высоте окна

## 📊 Технические детали

### Вычисление позиции скролла:
```typescript
// Абсолютная позиция элемента от начала страницы
const absoluteElementTop = rect.top + window.pageYOffset;

// Позиция для центрирования элемента по вертикали
const scrollToY = absoluteElementTop - (window.innerHeight / 2) + (rect.height / 2);

// Ограничение (не прокручивать выше начала страницы)
const finalScrollY = Math.max(0, scrollToY);
```

### Последовательность действий:
```
1. Определить текущий шаг tour
2. Найти целевой элемент (querySelector)
3. Получить позицию элемента (getBoundingClientRect)
4. Проверить видимость элемента
5. Если не виден:
   a. Вычислить позицию скролла
   b. Плавно прокрутить (smooth)
   c. Подождать 600ms
   d. Обновить позицию tooltip
6. Если виден:
   a. Сразу позиционировать tooltip
```
