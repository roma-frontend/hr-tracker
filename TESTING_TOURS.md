## 🎯 Тестирование Login Tour

### Подготовка:
```javascript
// Откройте консоль браузера (F12) и выполните:
localStorage.removeItem('tour_seen_login-tour');
// Или откройте в режиме инкогнито
```

### Шаги тестирования:

#### Шаг 1: Приветствие
```
URL: http://localhost:3000/login

✅ Проверить:
- Tour запустился автоматически?
- Появился темный оверлей?
- Видна подсказка по центру?
- Текст: "👋 Добро пожаловать!"
- Описание на русском языке?
```

#### Шаг 2: Биометрия
```
✅ Проверить:
- Подсветка элемента #biometric-login
- Заголовок: "🔐 Быстрый вход"
- Описание: "Используйте отпечаток пальца или Face ID..."
- Стрелка указывает на элемент?
```

#### Шаг 3: Форма Email
```
✅ Проверить:
- Подсветка #email-login-form
- Заголовок: "📧 Умный ввод Email"

- Кнопки навигации работают?
```

#### Шаг 4: Автопроверка Email (НОВЫЙ!)
```
✅ Проверить:
- Подсветка поля ввода email
- Заголовок: "✨ Автопроверка Email"
- Описание: "gmial.com → gmail.com"

```

#### Шаг 5: Безопасный пароль (НОВЫЙ!)
```
✅ Проверить:
- Подсветка поля пароля

- Упоминание защиты данных?
```

#### Шаги 6-8: Навигация и финал
```
✅ Проверить:
- Шаг 6: Присоединиться к команде
- Шаг 7: Создать организацию
- Шаг 8: "🎉 Всё готово!"

### Интерактивность:
```
✅ Проверить:
- Кнопка "Далее" работает
- Кнопка "Назад" работает
- Кнопка "Пропустить" (X) работает

## 🎨 Визуальное тестирование

### Проверить анимации:
```
✅ Оверлей:
- Плавное появление темного фона
- Opacity корректный

✅ Подсветка элементов:
- Элемент выделяется светлой рамкой
- Остальное затемнено
- Анимация плавная

✅ Tooltip:
- Появляется рядом с элементом
- Стрелка указывает правильно
- Placement: top/bottom/left/right/center работает
```

### Проверить адаптивность:
```
✅ Desktop (1920x1080):
- Все элементы видны
- Tooltip не обрезается
- Подсветка корректна

✅ Tablet (768x1024):
- Tour адаптируется
- Текст читаем
- Кнопки кликабельны

✅ Mobile (375x667):
- Tour работает
- Элементы доступны
- Скролл работает
```

## 🔧 Отладка

### Если tour не запускается:

#### 1. Проверить localStorage:
```javascript
// В консоли:
localStorage.getItem('tour_seen_login-tour');
localStorage.getItem('tour_seen_register-tour');

// Если "true" - сбросить:
localStorage.removeItem('tour_seen_login-tour');
localStorage.removeItem('tour_seen_register-tour');
```

#### 2. Проверить консоль:
```javascript
// Должны увидеть:
"🤖 Tour started: login-tour" // или register-tour
"Current step: 0"
// и т.д.
```

#### 3. Проверить элементы на странице:
```javascript
// В консоли:
document.querySelector('#login-card'); // должен вернуть элемент
document.querySelector('#email-field');
document.querySelector('#password-field');

// Если null - элемент не найден, проверить ID
```
