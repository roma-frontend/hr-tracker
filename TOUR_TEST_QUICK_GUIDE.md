## 🎯 Что проверить

- Темный оверлей появляется?

- Тексты на русском?
- Эмодзи отображаются?

### ✅ Интерактивно:
- Кнопка "Далее" работает?
- Кнопка "Назад" работает?
- Кнопка "Пропустить" (X) закрывает?
- ESC закрывает tour?

### ✅ Функционально:
- Индикатор шагов обновляется (1/8, 2/8...)?

## 🐛 Если что-то не работает

### Tour не запускается:
```javascript
// Проверьте:
localStorage.getItem('tour_seen_login-tour');
// Если "true" - сбросьте:
localStorage.clear();
location.reload();
```

### Элемент не подсвечивается:
```javascript
// Проверьте наличие элемента:
document.querySelector('#email-field');
document.querySelector('#password-field');
// Если null - элемент не существует
```

### Консольные ошибки:
- Откройте F12 → Console
- Ищите красные ошибки
- Скопируйте и сохраните для отчета
